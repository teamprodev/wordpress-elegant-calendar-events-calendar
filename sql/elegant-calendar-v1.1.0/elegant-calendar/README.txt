Elegant Calendar is a flexible plugin that make you easily manage and share events.

== Description ==

https://www.youtube.com/watch?v=VFV05Uy1Ssk

Elegant Calendar make you can easily create and manage events calendar on your website. This plugin comes with built in options that you can use to generate different calendar layout as you needed.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Elegant Calendar` from Plugins page.
